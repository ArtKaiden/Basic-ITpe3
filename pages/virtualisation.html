Tijdens de lessen basic-IT skills leerden we wat virtualisatie is en hoe we een VM opstellen. Hier ga ik daar even over praten.

Wat is virtualisatie?

Virtualisatie is een technologie die het mogelijk maakt om meerdere instanties van een besturingssysteem te draaien op 1 computer. De instanties staan bijna volledig onafhankelijk van de host van de computer.

Waarom zou ik een virtual machine aanmaken?

Een virtual machine stelt je in staat om verschillende besturingssystemen te draaien op dezelfde hardware, waardoor je gemakkelijk software kunt testen, isoleren of specifieke taken kunt uitvoeren zonder de hoofdmachine te be√Ønvloeden. Als developer kun je hierbij ook al je software uitproberen op andere besturingssystemen. Ook enorm interessant is de mogelijkheid om veilig riskante dingen uit te proberen.

Welke programma's gebruik ik hiervoor?

Er zijn verschillende programma's beschikbaar, zoals VMware, VirtualBox en Hyper-V.
